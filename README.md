# AI 书签管家 (BookmarkMind AI)

> AI 驱动的 Chrome 书签管理扩展 — 收藏即整理，搜索即对话

**仓库地址：** https://github.com/shawn2016/bookmarkmind-ai

## 项目结构

```
.
├── bookmarkmind-ai/     # Chrome 扩展源码（React + TypeScript + Vite）
├── README.md            # 本文件
└── docs/                # PRD、架构设计等文档
```

扩展开发与构建说明见 [`bookmarkmind-ai/README.md`](bookmarkmind-ai/README.md)。

## 快速开始

```bash
cd bookmarkmind-ai
npm install
npm run build
```

构建后在 Chrome 加载 `bookmarkmind-ai/dist/` 目录即可。
