## 1. 设计原则

### 1.1 核心理念

| 原则 | 说明 |
|------|------|
| **轻量优先** | 悬浮球是入口，不是主界面。默认状态 30% 透明度，不打扰用户浏览 |
| **渐进式展开** | 悬浮球 → 迷你操作栏 → 完整面板，三级渐进，按需展开 |
| **对话即操作** | AI 对话框是核心交互方式，搜索/分类/批量操作都可通过自然语言完成 |
| **即时反馈** | 所有操作必须有即时视觉反馈（Toast/动画/状态变化），不超过 200ms |
| **Shadow DOM 隔离** | 所有注入页面的 UI 均使用 Shadow DOM，确保与宿主页面零样式冲突 |

### 1.2 尺寸约束

| 组件 | 尺寸 | 说明 |
|------|------|------|
| 悬浮球 | 48 × 48 px | 固定不可缩放 |
| 迷你操作栏 | 160 × 48 px | 悬停展开 |
| 悬浮面板（默认） | 380 × 520 px | 可调整 |
| 悬浮面板（最小） | 320 × 400 px | 最小约束 |
| 悬浮面板（最大） | 600 × 800 px | 最大约束 |
| 设置页 | 100% × 100% | 新标签页全屏 |

```css
:root {
```

## 4. 悬浮球组件规格

### 4.1 组件概览

悬浮球是整个扩展的入口，有三种视觉状态：**收起态 → 悬停态 → 展开态（面板）**。

### 4.2 收起态（Collapsed）

```
┌──────────────────────────────────┐
│                                  │
│         ┌──────────┐             │
│         │          │             │
│         │   🔖     │  ← 48×48   │
│         │          │             │
│         └──────────┘             │
│              ↕ 可拖拽             │
│         (固定在页面右侧边缘)       │
│                                  │
│  透明度: 30%（默认，可配置 0-100）│
│  圆角: 12px（圆角方形）            │
│  背景: rgba(255,255,255,0.9)     │
│  边框: 1px solid --bm-gray-200   │
│  阴影: --bm-shadow-ball          │
│  图标: 24×24，颜色 --bm-primary-500│
│                                  │
└──────────────────────────────────┘
```

**视觉规格**：

| 属性 | 值 | 说明 |
|------|-----|------|
| 尺寸 | 48 × 48 px | 固定不变 |
| 形状 | 圆角方形（rx=12px） | 默认风格；可选圆形（rx=9999px） |
| 背景色 | `rgba(255, 255, 255, 0.9)` | 半透明白底 |
| 边框 | `1px solid var(--bm-gray-200)` | 浅灰边框 |
| 阴影 | `var(--bm-shadow-ball)` | 轻微投影 |
| 透明度 | `0.3`（默认） | `opacity: 0.3`，可配置 0~1 |
| 图标尺寸 | 24 × 24 px | 居中放置 |
| 图标颜色 | `var(--bm-primary-500)` | 主色 |
| 位置 | 右侧边缘，距右边 0px | 固定 `position: fixed; right: 0` |
| 垂直位置 | 距顶部 50% | `top: 50%; transform: translateY(-50%)` |
| 拖拽 | 支持垂直拖拽 | 拖拽时显示抓手光标 |

### 4.3 悬停态（Hover）

```
┌──────────────────────────────────┐
│                                  │
│    ┌──┬──┬──┬──┐┌──────────┐    │
│    │🔖 │🔍│⚙️ │📂││          │    │
│    └──┴──┴──┴──┘│   🔖     │    │
│    ← 迷你操作栏  │          │    │
│      (160×48)   └──────────┘    │
│                                  │
│  透明度: 100%（恢复不透明）        │
│  阴影: --bm-shadow-ball-hover    │
│  缩放: scale(1.05)               │
│  过渡: 150ms ease-out            │
│                                  │
└──────────────────────────────────┘
```

**状态变化**：

| 属性 | 收起态 → 悬停态 | 过渡 |
|------|-----------------|------|
| 透明度 | 0.3 → 1.0 | `150ms ease-out` |
| 阴影 | `--bm-shadow-ball` → `--bm-shadow-ball-hover` | `150ms` |
| 缩放 | `scale(1)` → `scale(1.05)` | `150ms ease-spring` |
| 操作栏 | 隐藏 → 展开出现 | `250ms ease-out`，从悬浮球左侧滑入 |

**触发规则**：
- 鼠标移入悬浮球区域 → 300ms 延迟后展开操作栏
- 鼠标移出悬浮球+操作栏区域 → 500ms 延迟后收起操作栏
- 操作栏出现时悬浮球透明度恢复为 100%

### 4.4 拖拽交互

| 属性 | 值 |
|------|-----|
| 拖拽手柄 | 悬浮球本体 |
| 拖拽方向 | 垂直（仅上下） |
| 拖拽范围 | 视口顶部 20px ~ 视口底部 - 20px |
| 拖拽时透明度 | 0.8 |
| 拖拽时光标 | `cursor: grabbing` |
| 拖拽时缩放 | `scale(0.95)` |
| 释放后 | 位置记忆到 `chrome.storage.local` |
| 贴边吸附 | 距边缘 < 10px 时吸附到边缘 |
| 长按 | 不触发任何操作（仅拖拽） |

### 4.5 右键菜单

| 菜单项 | 说明 |
|--------|------|
| 在当前网站隐藏悬浮球 | 添加当前域名到禁用列表 |
| 展开面板 | 等同点击行为 |
| 设置 | 打开设置页 |
| 关于 | 显示版本信息 |

### 4.6 自动隐藏规则

| 条件 | 行为 |
|------|------|
| 页面进入全屏模式（Fullscreen API） | 悬浮球淡出隐藏，退出全屏后恢复 |
| 检测到 `<video>` 全屏播放 | 同上 |
| Chrome 内置页面（chrome://） | 不注入 |
| PDF 页面（Content-Type: application/pdf） | 不注入 |
| 编辑器类网站（Figma, CodePen, JSFiddle） | 默认不注入，可在设置中开启 |
| CSP 阻止注入 | 降级为工具栏 Popup |

## 5. 迷你操作栏组件规格

### 5.1 布局

```
┌──────────────────────────────────────────┐
│                                          │
│    ┌─────────────────┐  ┌──────────┐     │
│    │                 │  │          │     │

│    │                 │  │          │     │
│    └─────────────────┘  └──────────┘     │
│     ← 操作栏 160×48       ← 悬浮球 48×48 │
│     gap: 4px                             │
│                                          │
│  从悬浮球左侧弹出                          │
│  出现动画: translateX(20px→0) + opacity   │
│                                          │
└──────────────────────────────────────────┘
```

### 5.2 视觉规格

| 属性 | 值 |
|------|-----|
| 尺寸 | 160 × 48 px |
| 背景 | `var(--bm-gray-0)` (白色) |
| 边框 | `1px solid var(--bm-gray-200)` |
| 圆角 | `var(--bm-radius-lg)` (12px) |
| 阴影 | `var(--bm-shadow-card)` |
| 内边距 | `0 var(--bm-space-2)` |

| 与悬浮球间距 | 4px |
| 出现动画 | `translateX(20px) → 0` + `opacity 0 → 1`，`250ms ease-out` |
| 消失动画 | `translateX(0) → 20px` + `opacity 1 → 0`，`200ms ease-in` |

### 5.3 操作按钮

| 按钮顺序 | 图标 | 功能 | Tooltip | 快捷键 |
|----------|------|------|---------|--------|
| 1 | 书签图标 (🔖) | 收藏当前页 | "收藏此页" | `Alt+S` |
| 2 | 搜索图标 (🔍) | 聚焦 AI 对话框 | "AI 搜索" | `Alt+F` |
| 3 | 设置图标 (⚙️) | 打开设置页 | "设置" | — |
| 4 | 展开图标 (⤢) | 展开完整面板 | "展开面板" | `Alt+B` |

**按钮通用规格**：

| 属性 | 值 |

## 16. 动效规范

### 16.1 动效原则

| 原则 | 说明 |
|------|------|
| **快速响应** | 所有交互反馈 ≤ 200ms |
| **自然缓动** | 使用 `ease-out` 出现、`ease-in` 消失、`ease-spring` 弹性 |
| **不滥用** | 仅在状态变化时使用动效，不为装饰而动画 |
| **可降级** | 尊重 `prefers-reduced-motion`，关闭非必要动画 |

### 16.2 标准动效清单

| 动效名 | 触发场景 | 属性变化 | 时长 | 缓动 |
|--------|---------|----------|------|------|
| 淡入 | 面板展开、Modal 出现 | `opacity 0→1` | 200ms | ease-out |
| 淡出 | 面板收起、Modal 消失 | `opacity 1→0` | 150ms | ease-in |
| 缩放出现 | Modal | `scale(0.95)→1` | 250ms | ease-spring |
| 滑入 | Toast、操作栏 | `translateX/Y(20px)→0` | 200ms | ease-out |
| 滑出 | Toast、操作栏 | `translateX/Y(0)→20px` | 150ms | ease-in |
| 按钮按下 | 所有按钮 | `scale(1)→0.98` | 100ms | ease |
| Hover 上浮 | 悬浮球、卡片 | `scale(1)→1.05` | 150ms | ease-spring |
| 加载旋转 | Spinner | `rotate 0→360deg` | 800ms | linear |
| 骨架扫光 | Skeleton | `translateX(-100%)→100%` | 1.5s | ease |
| 打字光标 | AI 流式输出 | `opacity 1→0→1` | 530ms | ease |

### 16.3 降级处理

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```
